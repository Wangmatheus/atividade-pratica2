TechNova Soluções Digitais

Este projeto foi desenvolvido como parte de uma atividade prática de desenvolvimento web, com o objetivo de criar um site institucional simples utilizando HTML e CSS.

Sobre o projeto

O site apresenta a empresa fictícia TechNova Soluções Digitais, destacando seus principais serviços, informações institucionais e um formulário de contato para comunicação com clientes.

Páginas desenvolvidas

- **index.html** → Página inicial com apresentação da empresa  
- **servicos.html** → Lista de serviços e tabela de preços  
- **contato.html** → Formulário de contato com o cliente  

Tecnologias utilizadas

- HTML  
- CSS  

Organização do projeto

O projeto segue uma estrutura organizada de arquivos, separando páginas, estilos e imagens em diretórios específicos, facilitando a manutenção e compreensão do código.

Objetivo

Praticar conceitos fundamentais de desenvolvimento web, como estruturação de páginas HTML, aplicação de estilos com CSS (inline, interno e externo) e organização de projetos.



Desenvolvido para fins acadêmicos.
